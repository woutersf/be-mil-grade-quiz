Interactive quiz to learn Belgian Defence military ranks across Land, Air, and Marine components.

## Features

- **69 Belgian military ranks** (Land, Air, Marine components)
- **Component selection** - Choose which to practice
- **Three quiz modes** - 10, 20, or all questions
- **Instant feedback** - Learn from your mistakes
- **Mobile friendly** - Works on any device

## Files

- `index.html` - Main page
- `style.css` - Styling
- `app.js` - Game logic
- `ranks-data.js` - Rank data with insignia images

## Local Development

Open `index.html` in your browser. No build process required.

## Data Source

Rank insignia from [Wikimedia Commons](https://nl.wikipedia.org/wiki/Lijst_van_militaire_rangen_van_de_Belgische_strijdkrachten)
