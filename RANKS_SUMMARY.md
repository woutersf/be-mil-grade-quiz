# Belgian Military Ranks - Complete Data Summary

This document provides a comprehensive overview of all Belgian military ranks extracted from the source files, organized by component with NATO codes and insignia images.

## Data Structure

The data is organized into three components:
- **Land Component** (Landmacht/Army) - 23 ranks total
- **Air Component** (Luchtmacht/Air Force) - 23 ranks total
- **Marine Component** (Marine/Navy) - 23 ranks total

## Notes

- All images are hosted on Wikimedia Commons at 120px width for optimal display quality
- Some NATO codes (OF-1, OF-3, OR-4, OR-6, OR-9) have multiple rank variations
- The OR-1 rank (lowest enlisted) uses the same insignia across all three components
- Belgium does not use NATO rank OF-10
- Marine ranks use distinct naval terminology (e.g., Admiraal, Kapitein-ter-zee, Matroos)
- Land and Air components share similar rank naming conventions for officers and NCOs
